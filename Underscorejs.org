* Underscore.js
** About
- JavaScript library that provides a whole mess of useful functional programming helpers without extending any built-in objects.
- 提供しているメソッドをブラウザがサポートしている場合はネイティブの機能を利用し、サポートしていないブラウザにも同等の機能を与える。
