**** Replace
- 
  Ctrl + f -> "Find" Window
  Alt + r  -> Move to "Replace" Window

  Alt + f  -> "Find Next"
  Alt + r  -> "Replace"

**** Grid
- 
  View Tab -> Chcek "Gridlines" checkbox

- show/hide guide : F9
- add guide : Ctrl + Daug
- show relative position from current : Shift + Drag
- delete : move away a line to outside of slide
